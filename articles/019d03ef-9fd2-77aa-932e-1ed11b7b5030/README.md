## メモ

たとえば家庭用ゲーム機において、分解・解析・改造を禁止する利用規約が広く存在する。しかし対価と引き換えにユーザーの手元に完全に渡ったハードウェアや、それに選択の余地なく付随するソフトウェアについて、その利用状況を追跡する確実な方法は存在せず、また追跡すべきでもない。オンライン接続などは一切行わず、規約と無関係に犯罪に該当する行動をしない当然の前提をおけば、違反を検知することも制限の執行も行えない規約は一方的な行動制限の強要であると断じるほかなく、なおたちの悪いことに契約としては有効だ。

別の例として、フォントサブスクリプションサービスにおいては、完全なフォントファイルをユーザーのローカル環境に配置せざるを得ない（これはフォントシステムが発展する過程でDRMを織り込まなかったという技術的な事情による。フォントファイルを暗号化したりインメモリに展開する機能は、少なくとも現在のところ、フォントを利用する多くのアプリケーションで一般的とはいえない）。フォントファイルを外部に公開するなど明らかに知財を侵害する行為でもなければ、展開されたフォントを利用したか・コピーしたフォントを利用したかを区別して制限を執行することは技術的に不可能だ。

以上の2つの例を、技術的に制御不能なものを契約だけで制御しようとすること自体の正当性の議論につなげたい。一般の利用者は自身が違反していないことを証明し続けなければ規約違反のリスクを負い、にもかかわらず提供者はその証明すら確実に検証することはできないという非対称な構造がある。そして一般の利用者が法廷闘争に持ち込むコストは提供者が利用者を訴えるコストに比べて異常に高く、事実上争うことすらできない。

実効性のない自制の要請と、強制力のある契約を分離すべきだ。提供者には実態が制御不能になってなお利用者を脅迫してまで保全したい資産があるように、利用者にも自身のレピュテーションを維持する必要性や提供者との共栄のために利益調整をおこなう意思がある。誰だって犯罪者の誹りを受けたいわけではないし、まともな社会経験があれば自分の仕事が多くの関係者との利害関係で成り立っていることを理解している。無理筋で契約という形を取らずとも、道徳的に合意可能な均衡は確かに存在する。

仮にフォントサブスクリプションサービスがこの枠組みの中で滅びてしまうというなら、それは利用者と制作者の間に共栄の意思がなかったことが明らかになるということだろう。ならば別のモデルを検討すべきであり、実効性のない規約が契約であるか道徳的な通知に過ぎないのかとは関係がない。たとえばフォント暗号化の標準化をすすめるためのコストや対応アプリのみで利用できる制限を行うコスト、フリーライダーを排除するコスト、これら利用者と提供者が本来分割して負うべきコストを、利用者の無関心につけこんで悪魔の証明を強いることで転嫁しているに過ぎない。

利用者はきっとあらゆるものを安価に享受しすぎたのだろう。その結果とんでもないコストを負わされていることに、利用者はまだ気づけないでいる。われわれはコスト配分を再設計する必要がある。われわれはじめから共栄関係にあるのであり、提供者は利用者にそれを自覚させる必要がある。他方利用者は、自らが選んだ相手と関係を維持するコストを積極的に払う必要がある。それは、いつでも利用者を疑い排除できる契約である利用規約ではなく、実直な要請と相互の利益調整によってこそ実現される善意の関係性にほかならない。
